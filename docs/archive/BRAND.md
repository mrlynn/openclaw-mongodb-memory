Primary palette

These colors define the core MongoDB brand. Use them consistently across product, marketing, events, and digital experiences.

Spring Green
• Hex: #00ED64
• CMYK: 57 / 0 / 79 / 0
• Pantone formula:
• Pantone Green: 28.35
• Pantone Yellow 012: 6.56
• Pantone Transparent White: 65.09
• Closest existing PMS color: 902C

Usage guidance:
• Primary brand accent.
• Calls to action.
• Highlights and key UI elements.
• Do not use for long-form body text on light backgrounds due to contrast limits.

⸻

Forest Green
• Hex: #00684A
• PMS: 7733 C
• CMYK: 89 / 32 / 89 / 22

Usage guidance:
• Primary dark green.
• Section backgrounds.
• Headers and strong emphasis areas.
• Works well with White and Mist for accessible text.

⸻

Evergreen
• Hex: #023430
• PMS: 553 C
• CMYK: 78 / 48 / 76 / 49

Usage guidance:
• Deep brand anchor.
• Dark backgrounds.
• Navigation bars.
• High-contrast layouts with White, Mist, and Lavender.

⸻

Slate Blue
• Hex: #001E2B
• PMS: 296 C
• CMYK: 89 / 75 / 55 / 69

Usage guidance:

⸻

Shades and tints strategy

MongoDB does not support a full spectrum of arbitrary tints for each color.

Instead:
• Use the predefined secondary and mid-tone colors.
• Select from approved light tints such as Mist, Lavender, Lime, Azure, Lilac, and Glow.
• Keep combinations consistent across touchpoints.

A swatch file for Adobe products should be distributed internally to ensure accuracy.

⸻

Accessibility guidance

MongoDB follows WCAG 2.1 AA standards.

Minimum contrast ratio:
• 4.5:1 for standard text.
• 3:1 for large-scale text, defined as 18pt regular or 14pt bold.

Approved high-contrast combinations

These pairings meet or exceed 4.5:1:
• Spring Green on Forest Green
• Spring Green on Evergreen
• Spring Green on Slate Blue
• White on Forest Green
• White on Evergreen
• White on Slate Blue
• Forest Green on White
• Evergreen on White
• Slate Blue on White
• Forest Green on Mist
• Evergreen on Mist
• Slate Blue on Mist

Avoid:
• Spring Green on White
• Spring Green on Mist
• Spring Green on Lavender
• Light colors on White
• Mist or Lavender text on White

Design rule:
If contrast is uncertain, test the exact foreground and background hex values with a WCAG contrast checker before release.

⸻

Implementation checklist

Before shipping any design:
• Confirm hex values match the approved palette.
• Validate contrast ratios for all text.
• Limit color combinations to approved sets.
• Avoid creating unapproved tints.
• Keep Spring Green as a brand signal, not a body text color.
